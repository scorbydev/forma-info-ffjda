# Prerequis J2

- Codex App ou CLI operationnel.
- DB Browser for SQLite installe.
- Python disponible.
- Node.js et npm disponibles.
- Compte Vercel personnel pret si possible.
- Vercel CLI installable.
- Vercel MCP est l'atelier principal, avec fallback si OAuth bloque.

Verification:

```bash
python --version
node --version
npm --version
```

Vercel CLI:

```bash
npm i -g vercel
vercel --version
```

Vercel MCP:

```bash
npx add-mcp https://mcp.vercel.com
```

Ne jamais copier un token Vercel dans le repo ou dans un prompt.
