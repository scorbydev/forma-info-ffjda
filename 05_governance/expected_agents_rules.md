# Regles AGENTS attendues

Un bon `AGENTS.md` doit contenir:

- inspection initiale obligatoire;
- plan avant modification;
- donnees reelles interdites;
- secrets interdits dans le code;
- `.env` local et `.env.example`;
- validation humaine avant action destructive;
- SQL en diagnostic puis transaction;
- tests obligatoires pour le code;
- OCR avec OpenRouter optionnel, Hugging Face fallback et sample offline;
- documentation des risques et limites;
- resume final avec fichiers modifies, tests et limites.
