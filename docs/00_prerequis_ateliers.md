# Prerequis ateliers

- Installer Codex App ou Codex CLI.
- Installer Antigravity App ou CLI comme fallback.
- Verifier Git: `git --version`.
- Verifier Python: `python --version`.
- Creer ou preparer un compte ChatGPT.
- Preparer un compte Gmail pour le fallback Antigravity.
- Creer un compte OpenRouter optionnel pour l'atelier OCR.
- Creer un compte Hugging Face pour le fallback OCR.
- Copier `.env.example` vers `.env` si une API est utilisee.
- Ne jamais commiter `.env`.
